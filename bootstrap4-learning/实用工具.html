Bootstrap4 实用工具

在Bootstrap4中包含了很多实用的小工具，这些功能能够让你不用写 CSS 代码就能实现想要的效果，让开发更快捷和简单。

边框

使用 border 类可以添加或移除边框:

class="border"

class="border border-0"

class="border border-top-0"

class="border border-right-0"

class="border border-bottom-0"

class="border border-left-0

边框颜色

Bootstrap4 提供了一些类来设置边框颜色:

class="border border-primary"

class="border border-secondary"

class="border border-success"

lass="border border-danger"

lass="border border-warning"

class="border border-info"

class="border border-light"

class="border border-dark"

lass="border border-white"

边框圆角设置

使用rounded 类可以添加圆角边框。

下列使用的边框样式分别为：rounded，rounded-top，rounded-right，rounded-bottom，rounded-left，rounded-circle，rounded-0。

全部圆角 上方圆角 右边圆角 下方圆角 左边圆角 原型图片 没有圆角

浮动

.float-right 类用于设置元素右浮动， .float-left 设置元素左浮动, .clearfix 类用于清除浮动:

左浮动 右浮动

响应式浮动

重置浏览器大小查看效果。

我们可以设置浮动 (.float-*-left|right - * 为 sm, md, lg 或 xl)的方向依赖于屏幕的大小:

在大于小屏幕尺寸上右浮动

在大于中等屏幕尺寸上右浮动

在大于大屏幕尺寸上右浮动

在大于超大屏幕尺寸上右浮动

没有浮动
